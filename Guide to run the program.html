Guide to run the program

Back-end

1.use any python IDE you like(Python 3.7)

2.Run cmd.exe

3. install relevant package

pip install flask,flask_mysqldb

4.create database

The field name and its format should be like this

Field Type Null Key Default Extra
id int(10) unsigned NO PRI NULL auto_increment
name varchar(30) NO NULL
email varchar(30) NO NULL
city varchar(20) NO NULL
phone varchar(30) NO NULL

5.Run Back-end

.\CourseWork2\CMD.exe

python api.py

Front-end

1. install Vue.js

.\CourseWork2\Front-end\client\cmd.exe

npm install vue

2. install relevant packages

.\CourseWork2\Front-end\client\cmd.exe

 npm install

3. run Front-ed

npm run serve
